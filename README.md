# Hello World Welcome to my Habit tracker

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install My Habit tracker.

Use node v20 use :
```bash
nvm install 20
```

```bash
npm i
```

## Usage

Start the application dev with :

```bash
npm run start
```

Created the dist with :

```bash
npm run dist
```

Analyse the coding rules with :

```bash
npm run lint
```
